Creating polygons

Neuorglancer provides the capablity to create multiple types of annotations. The Brainsharer version of Neuroglancer extends these capabilities to include the creation of polygons. These polygons can be used to create 3D contours of cytoarchitecture artifacts. The following steps describe how to create polygons in Neuroglancer and how to use them.

Logging in and finding your data

  • Login to brainsharer.org Admin page (upper right-hand corner of page):
  • After login, go back to login area and select ‘Admin’
  • In the admin portal, navigate to the Neuroglancer section and select ‘Neuroglancer states’
  • Use top search tool to locate Neuroglancer states by typing brain name (animal id):
  • Click on the desired link under the ‘NEUROGLANCER’ column to open this brain state in Neuroglancer

Creating polygons

  • You will need to create a new layer. Do this by holding the CTRL key and clicking the '+' icon just right of the right most open layer.
  • In the upper right part the window, click the annotations tab.
  • There are 7 types of different annotation icons. Click the square icon. Hovering over this you should see a 'Annotate volume' help text.
  • A volume is simply a set of polygons that will create the 3D outline of the brain region you are working on.
  • You will want to add polygons using just one quadrant, so click the maximize icon in the upper left quadrant view.
  • Using the 'Z' dropdown menu near the top left of the window, go to the z section you want to start on.
  • Using 'CTRL' click, start adding vertices to complete a polygon on the current section. Click 'enter' to automatically close the polygon.
  • Create one polygon per section for an individual volume. Carry on this process on each section you want to create a polygon on.
  • Remember to save the Neuroglancer state as you go along by clicking the 'Save' icon in the upper left side panel.
