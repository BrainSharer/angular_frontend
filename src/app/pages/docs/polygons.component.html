3D contour creation from annotations

3D contour creations may be constructed from polygon annotations and superimposed on existing brain images. Prior to creating them, you will need to create polygon annotations around the region of interest (ROI). To create more accurate 3D structures, you should create polygons with many vertices, especially around edges.

Process and example

Create annotations

  • Login to brainsharer.org Admin page (upper right-hand corner of page):
  • After login, go back to login area and select ‘Admin’
  • In the admin portal, navigate to the Neuroglancer section and select ‘Neuroglancer states’
  • Use top search tool to locate Neuroglancer states by typing brain name (animal id):
  • Click on link in ‘Neuroglancer’ column to open this brain state in Neuroglancer (smaller box in figure above)
  • Create polygons on the sections that contain cytoarchitecture artifacts of interest. Polygon creation is described on: https://webdev.dk.ucsd.edu/docs/brainsharer/neuroglancer/modules/polygon_tool.html

Previw and save annotations

  • Once the polygons are created with sufficient detail, go to the layer (top tabs) where they are listed and select ‘Annotations’ subtab (upper-right corner of screen):
  • Select annotation set of interest. in figure above, TG_L is selected. When you roll your mouse cursor over the annotation set that contains the polygons to be used for the 3D contour creation, you may see an arrow icon (pointing down) to represent that this annotation set may be used for creating a 3D contour.

    • Note: If this icon does not appear, you will need to triage just those polygons from larger ‘Neuroglancer state’. This is accomplished by creating a new annotation set in database through the following steps: 1) with polygons selected in top panel, click on ‘new’ button in lower panel. 2) A notification in lower-left will indicate when the creation is complete.

Generate 3D mesh/contour view

  • Returning to upper-right annotations panel, click the arrow icon (pointing down). This will generate a 3D view of the stacked polygons, using interpolation. Notification in lower-left will say ‘creating 3d mesh’.

    • Note: A new layer tab will appear at top, next to existing annotation/image tabs. This segmentation layer will have the new 3D contour/mesh.

  • To view the 3D contour/mesh in display panels, you will need to toggle visibility. To accomplish this, select ‘Seg.’ tab in upper part of the new 3D contour layer. Below selection, you will see a cross-out eye icon which must be clicked to show the 3D contour/mesh.

    • Note: You may wish view multiple overlays on a single region of interest. For better visibility, you may adjust opacity of annotations and/or contours. For the contour/mesh volumes, the adjustments may be found on the ‘Render’ tab. The settings that affect contours are the ones ending in ‘3d’.
