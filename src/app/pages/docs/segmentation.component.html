Creating 3D segmentations/meshes in Brainsharer Neuroglancer

Brainsharer Neuroglancer allows you to create 3D segmentations/meshes from the annotation data. This is a step-by-step guide to create a 3D mesh from the annotation data.

This process will be simplified once the UI is finished.

Exporting volumes
  • Go to the Neuroglancer view: e.g., https://brainsharer.org/ng/?id=945 .
  • Select annotation layer with control click.
  • Make sure each volume you want to export is labeled.
  • Click on a volume and go to the selection area in the lower right.
  • Click the 'new' icon. This will export the volume data to the database.
Find exported annotation IDs
  • Go to: https://brainsharer.org/brainsharer/admin/neuroglancer/annotationdata/ The data is sorted with the most recentlty exported data at the top.
  • Find the ID of the annotation you just exported. For this example it is 999.
  • Type this in the URL location area of your browser: https://brainsharer.org/brainsharer/annotations/segmentation/999
  • This will take 10 to 60 seconds to complete. At the end you will see a brief response like: 
    "url": "precomputed://https://brainsharer.org/structures/CTB004_SC_Gray_Matter", "name": "CTB004_SC_Gray_Matter"
  • Copy 'https://brainsharer.org/structures/CTB004_SC_Gray_Matter' . We will use it in a new layer in Neuroglancer
Create segmenation
  • Open up Neuroglancer and go back to your original view: https://brainsharer.org/ng/?id=945
  • Create a new image/mesh layer (NOT annotation layer) by clicking the '+' icon at the top of the view.
  • In the precomputed load area, paste the URL you copied earlier.
  • This will load the mesh into the 3 quadrants.
  • To display the 3D mesh in the lower left quadrant:
  • Go to the 'Seg.' tab in the top right tab.
  • Toggle the 'eye' icon. The mesh should now appear in the lower left quadrant.
  • Repeat for each volume you want to export and display as a mesh.
