Creating 3D meshes/segmentions

3D contour creations may be constructed from polygon annotations and superimposed on existing brain images. Prior to creating the 3D contours, you will need to create polygon annotations around the region of interest (ROI). To create more accurate 3D structures, you should create polygons with many vertices, especially around edges.

Create annotations

  • Make sure you are logged in and you are viewing your existing set of polygons that contain cytoarchitecture artifacts of interest. Logging in and polgyon creation are described here: Creating polygons

Preview and save annotations

  • Once the polygons are created with sufficient detail, go to the layer (top tabs) where they are listed and select ‘Annotations’ subtab (upper-right corner of screen):
  • Select the annotation set of interest. In the figure above, TG_L is selected. When you roll your mouse cursor over the annotation set that contains the polygons to be used for the 3D contour creation, you should see an arrow icon (pointing down) to represent that this annotation set may be used for creating a 3D contour.

    • Note: If this icon does not appear, make sure you are logged in and that your data has been labeled (this data has been labeled TG_L) and exported. This is accomplished by creating a new annotation set in database through the following steps:

    • 1) with polygons selected in top panel, click on ‘new’ button in lower panel.
    • 2) A notification in lower-left will indicate when the creation is complete.

Generate 3D mesh/contour view

  • Returning to upper-right annotations panel, click the arrow icon (pointing down). This will display a small popup window where you can select the smoothing and interpolation options. The default options are usually sufficient. If you want no smoothing, set the x,y,z values to 0 and select 'No' for interpolation in the Z plane. Increasing the x,y,z values will result in a smoother contour/mesh. Click ‘Run’ to begin the process. The popup will close automatically and you will see a notification in the lower-left of the screen.

    • * Note: A new layer tab will appear at top, next to existing annotation/image tabs. This segmentation layer will have the new 3D contour/mesh.

    ** Note: If you adjust the smoothing variables multiple times, you'll need to first save the view with the 'Save' button and then reload the page. Your browser may cache the previous version of the contour/mesh and not show the updated version until the page is reloaded.

  • To view the 3D contour/mesh in display panels, you will need to toggle visibility. To accomplish this, select ‘Seg.’ tab in upper part of the new 3D contour layer. Below selection, you will see a cross-out eye icon which must be clicked to show the 3D contour/mesh.

    • Note: You may wish view multiple overlays on a single region of interest. For better visibility, you may adjust opacity of annotations and/or contours. For the contour/mesh volumes, the adjustments may be found on the ‘Render’ tab. The settings that affect contours are the ones ending in ‘3d’.
