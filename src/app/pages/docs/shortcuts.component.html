Polygon/Volume keyboard shortcuts

Neuroglancer has keyboard shortcuts available for creating polygons. Below is a list of the most commonly used keyboard shortcuts. Immediate help is also available from within Neuroglancer by simply clicking the 'h' key. This will toggle the help panel on and off in the left side of the window.

The following commands are useful when editing and adding vertices to polygons. This assumes the user is creating a polygon within a volume and is in 'Edit' mode.

  • CTRL click Adds new vertices
  • ALT shift moves the polygon
  • ENTER Completes and closes a polygon
  • SHIFT double click add vertex
  • SHIFT r Rotates polygon clockwise
  • SHIFT e Rotates polygon counter clockwise
  • SHIFT '=' (equals) Enlarge polygon
  • SHIFT '-' (minus) Shrink polygon
